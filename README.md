# 🧹 Invoice BersihBersih

Aplikasi invoice berbasis web (PWA) untuk jasa kebersihan profesional. Bisa diinstall langsung di HP tanpa Play Store.

---

## 🌐 Buka Aplikasi

👉 **[Klik di sini untuk membuka aplikasi](https://USERNAME.github.io/NAMA-REPO)**

> Ganti `USERNAME` dan `NAMA-REPO` sesuai akun GitHub kamu

---

## 📱 Install di HP (Android)

1. Buka link aplikasi di **Google Chrome**
2. Ketuk ikon **⋮** (tiga titik) di pojok kanan atas
3. Pilih **"Tambahkan ke layar utama"**
4. Ketuk **Tambahkan** → app siap digunakan

## 🍎 Install di iPhone (iOS)

1. Buka link aplikasi di **Safari**
2. Ketuk ikon **□↑** (share) di bawah layar
3. Gulir ke bawah → pilih **"Tambahkan ke Layar Utama"**
4. Ketuk **Tambahkan**

---

## 📥 Download Kode

Klik tombol **Code → Download ZIP** di halaman ini, atau clone:

```bash
git clone https://github.com/USERNAME/NAMA-REPO.git
```

Lalu buka file `index.html` langsung di browser — tidak perlu install apapun.

---

## ✨ Fitur

- 📄 Buat invoice dengan nomor otomatis
- 👤 Data pelanggan (nama, WhatsApp, alamat)
- 🧾 Tambah item layanan dengan qty & harga
- 💰 Kalkulasi otomatis (diskon, pajak, biaya tambahan)
- 📱 Kirim invoice langsung via WhatsApp
- 🖨️ Download sebagai PDF
- 🌙 Tema navy royal
- 📶 Bisa dipakai offline (PWA)

---

## 🛠️ Teknologi

- HTML, CSS, JavaScript murni (tanpa framework)
- Progressive Web App (PWA)
- GitHub Pages untuk hosting gratis
